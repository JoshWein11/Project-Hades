================================================================
       PROJECT HADES
  A 2D Top-Down Action Game
================================================================


DESCRIPTION
-----------
Project Hades is a 2D horror top-down action game built in C using
the Raylib library. You control a character from a bird's-eye
view, exploring the world, avoiding obstacles, and managing
your health as you fight or hide to survive.


HOW TO RUN
----------
1. Extract the ZIP file to any folder on your computer.
2. Open the Project Hades folder.
3. Double-click  game.exe  to start the game.

That's it! No installation is required.


CONTROLS
--------
  Move Up        W  or  Arrow Up
  Move Down      S  or  Arrow Down
  Move Left      A  or  Arrow Left
  Move Right     D  or  Arrow Right

  Open Setting   ESC

  Dash           Left Shift
                 (Hold a direction key while pressing Shift)

  Aim            Move your mouse cursor

  Quit           Close the window or press Alt + F4


FILES
-----
The folder should look like this after extracting:

  game.exe          <-- Double-click to launch the game
  assets/           <-- Contains all images, sounds, and data

IMPORTANT: The  assets  folder must stay in the same folder at all times. Moving or deleting it will cause the
game to fail to load.


REQUIREMENTS
------------
  Operating System:  Windows 10 or later (64-bit) !!
  No extra software or drivers are needed.


TROUBLESHOOTING
---------------
The game does not start:
  - Make sure you fully extracted the ZIP file before running.
    Right-click the ZIP and choose "Extract All", then open
    the extracted folder and run game.exe from there.
  - Make sure the Assets folder stay in the same folder from when
    extracting the file
  - If Windows shows a security warning, click "More info"
    then "Run anyway".

The game crashes immediately:
  - Make sure your Windows version is up to date.
  - Try running game.exe as Administrator (right-click ->
    "Run as administrator").
